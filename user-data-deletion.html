User Data Deletion

Socialmix is a self-hosted tool. The operator who runs the deployment controls the local database, video storage, Facebook Page token, and generated media files.

Remove Facebook Access

  1. Open Facebook Settings.
  2. Go to Business Integrations or Apps and Websites.
  3. Find Socialmix.
  4. Remove the app to revoke its access to your Facebook account or Page.

Delete Data From a Socialmix Deployment

To delete data stored by Socialmix, contact the app operator who runs the deployment and manages the connected Facebook Page. Ask them to delete your related videos, generated clips, transcripts, publish history, and stored Facebook access token.

Self-Hosted Operator Deletion Steps

If you are the deployment operator, remove the Facebook token from environment variables, delete related database records, and delete related files from the storage directory.

  • Remove FACEBOOK_PAGE_ACCESS_TOKEN from the deployment environment.
  • Delete related video, segment, subtitle, transcript, and thumbnail files.
  • Delete related database rows from the Socialmix database.
  • Restart the API and worker after changing environment variables.

Deletion Confirmation

The deployment operator should confirm deletion after the related local records and files have been removed.
